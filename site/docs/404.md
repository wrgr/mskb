# Page Not Found

If you entered a `.md` URL directly, this site uses pretty URLs.

Try the same path without `.md` and with a trailing slash.

<script>
(() => {
  const path = window.location.pathname || "";
  if (path.endsWith(".md")) {
    const redirected = path.replace(/\.md$/i, "/");
    window.location.replace(redirected + window.location.search + window.location.hash);
  }
})();
</script>
