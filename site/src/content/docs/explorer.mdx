<div class="explorer-shell mskb-viz-wide">

<p class="mskb-viz-tagline">Inspect papers, follow citation paths, and turn a short research note into parent/child/related paper choices.</p>

<p>Use this view to inspect citation structure, read paper summaries, and plan literature exploration from core papers out to related work. Papers are grouped by <strong>research themes</strong> derived from citation communities.</p>
<p><strong>Planning a multi-step study sequence?</strong> <a href="/mskb/journey/">Open in Learning Journey →</a></p>

<div class="explorer-guide" role="list" aria-label="Explorer usage guides">
  <div class="guide-card" role="listitem"><strong>Undergrad flow</strong> — set Reading level to <em>Basic</em> for plain-language summaries, then branch through related papers.</div>
  <div class="guide-card" role="listitem"><strong>Grad / clinician flow</strong> — set Reading level to <em>Advanced</em> for specialist wording. Use in-degree / k-core filters to surface high-signal, densely-cited papers.</div>
  <div class="guide-card" role="listitem"><strong>Node semantics</strong> — node size = log(citations), color = k-core tier, arrows = citations. Use <em>Cluster spread</em> if the graph looks too dense, <em>Re-Layout</em> to reset positions.</div>
</div>

  <p><strong>Reading level</strong> picks which summary variant to display for each paper. Every paper in the T1–T4 corpus (649 papers, April 2026) has a <em>Basic</em> (plain-language undergraduate) and an <em>Advanced</em> (specialist technical) summary; this toggle swaps between them in place. It does <strong>not</strong> hide papers from the graph.</p>
