## II. Learner Journey: Three Entry Points

### **Entry Point 1: Citation-First (Paper Explorer)**

```
Learner: "I found an interesting paper on B cells"
                  ↓
        [Paper Detail View]
        ├─ Title, DOI, Abstract
        ├─ Concept Tags: B Cell Biology, Therapeutics
        ├─ Citation network: Cites (10), Cited-by (50), Related (30)
        └─ Learning Hub (NEW): Videos, tools, related concepts
                  ↓
        [Concept Page] ← NEW LAYER 2
        ├─ Learning objectives
        ├─ Prerequisites (e.g., T Cell Biology)
        ├─ Key questions
        ├─ Recommended reading path
        └─ Resources (videos, tools, datasets)
                  ↓
        [Follow Path] ← NEW LAYER 3
        ├─ Watch: 15-min video on B cell depletion
        ├─ Read: Foundation paper + trial paper
        ├─ Use: DMT comparison calculator
        └─ Next: Explore "Treatment Monitoring"
```

### **Entry Point 2: Learning-First (Concept Pathway)**

```
Learner: "I want to learn about B cell therapies"
                  ↓
        [Concept Selection] ← NEW LAYER 2
        ├─ Browse hierarchical concept tree
        ├─ Filter by difficulty (Remember/Understand/Apply)
        ├─ Filter by pathways (clinical/mechanistic/cutting-edge)
        └─ Select: "B Cell-Targeted Therapies"
                  ↓
        [Concept Detail]
        ├─ Learning objectives: "After this, you'll understand..."
        ├─ Prerequisites: "Make sure you know about..."
        │  └─ [Link] T Cell and B Cell Biology → Learn first
        ├─ Key questions: "Ask yourself these..."
        └─ Resources ordered by level
                  ↓
        [Resource Consumption] ← NEW LAYER 3
        ├─ Foundation (Bloom's Understand):
        │  ├─ Video (10 min): B cell basics
        │  ├─ Review paper: B cells in MS
        │  └─ Tool: Concept map of mechanisms
        │
        ├─ Intermediate (Bloom's Apply):
        │  ├─ Paper: ASCLEPIOS trial
        │  ├─ Video: Clinical decision-making
        │  └─ Tool: DMT comparison calculator
        │
        └─ Advanced (Bloom's Analyze):
           ├─ Paper: EBV mimicry mechanistic study
           ├─ Dataset: Real-world outcomes (MSBase)
           └─ Video: Expert commentary on challenges
                  ↓
        [Next Recommendations]
        ├─ "You've learned B cell targeting → try BTK inhibitors"
        ├─ "Learn about treatment monitoring"
        └─ "Explore B cell biology papers you found"
```

### **Entry Point 3: Hybrid (Wander & Explore)**

```

## III. Architecture Layers: Data Model

```
LAYER 1: Citation Graph (Papers)
┌─────────────────────────────────────┐
│ Work                                │
│ ├─ DOI (unique identifier)          │
│ ├─ Title, Abstract                  │
│ ├─ Authors, Venue, Year             │
│ ├─ Citations (edges to other works)  │
│ └─ OpenAlex metadata                │
│    ├─ Concepts ([MS, Immunology])   │
│    ├─ Topics (from citation network)│
│    └─ Metrics (citations, PageRank) │
└─────────────────────────────────────┘

LAYER 2: Learning Concepts
┌─────────────────────────────────────┐
│ Concept                             │
│ ├─ ID (unique identifier)           │
│ ├─ Title (e.g., "B Cells in MS")    │
│ ├─ Bloom's level (Understand/Apply) │
│ ├─ Prerequisites (→ other concepts) │
│ ├─ Learning objectives (3-4 bullets)│
│ ├─ Key questions                    │
│ ├─ Related concepts (↔ other)       │
│ ├─ Papers (linked to Layer 1)       │
│ │  └─ 3-7 papers with roles:        │
│ │     (foundational, trial, mechanism)
│ └─ Evidence types (review/trial)    │
└─────────────────────────────────────┘

LAYER 3: Resources
┌─────────────────────────────────────┐
│ Resource (per Concept)              │
│ ├─ Type (video/tool/product/dataset)│
│ ├─ Title, URL                       │
│ ├─ Creator, Year                    │
│ ├─ Bloom's level (Remember/Apply)   │
│ ├─ Quality score (1-30)             │
│ ├─ Rationale (why this resource)    │
│ ├─ Strengths (what it does well)    │
│ ├─ Limitations (what it doesn't)    │
│ ├─ Curated by (who recommended)     │
│ └─ Last validated (when checked)    │
└─────────────────────────────────────┘
```

**Relationships**:
```
Concept ─(has_papers)─> Paper
Concept ─(has_prerequisite)─> Concept
Concept ─(has_resource)─> Resource
Resource ─(supports)─> Concept
```

## IV. Bias Detection System (What Gets Caught?)

```
BIAS DETECTION DASHBOARD
═══════════════════════════════════════════════════════════

🔴 RECENCY BIAS: Seeds only from last 3 years?
    ├─ FAIL if >80% papers from 2021+
    ├─ PASS if 60-70% recent + 30-40% foundational
    └─ FIX: Add landmark anchors from 1990s-2010s

🔴 VENUE BIAS: Same journal >3 papers?
    ├─ FAIL if any venue >3 papers
    ├─ PASS if diversity (>5 different venues)
    └─ FIX: Add papers from specialist journals

🔴 AUTHOR BIAS: Same first author >1 paper?
    ├─ FAIL if any author >1 seed paper
    ├─ PASS if each author appears once
    └─ FIX: Choose highest-quality paper per author

🔴 TOPIC BIAS: One mechanism >30% of seeds?
    ├─ FAIL if B cells = 50% but genetics = 5%
    ├─ PASS if mechanisms balanced (20-25% each)
    └─ FIX: Target underrepresented mechanisms

🔴 METHODOLOGICAL BIAS: Only RCTs, no observational?
    ├─ FAIL if 80% trials, 0% mechanistic
    ├─ PASS if: 25% trials + 25% mechanistic + 25% observational + 25% reviews
    └─ FIX: Explicitly add each methodology type

## VI. Success Indicators

### **Green Light ✓ (You've Succeeded)**

**Seed Quality**:
- ✓ Landmark set spans 1995-2022 (decade-balanced)
- ✓ <30% from any single journal
- ✓ >5 countries represented
- ✓ Governance checklist shows "PASS" with 0 errors

**Concept Coverage**:
- ✓ 20-25 core concepts defined and mapped
- ✓ All MS domains covered
- ✓ Each concept has 3-7 papers identified
- ✓ Prerequisite relationships documented

**Resource Diversity**:
- ✓ Each concept has ≥1 video, ≥1 tool/dataset
- ✓ Multiple authors represented across resources
- ✓ Mix of foundation + advanced + applied materials

**Learner Experience**:
- ✓ Students can start from 3+ different pathways
- ✓ Prerequisites are clear and testable
- ✓ Resources support each learning objective
- ✓ Navigation between papers/concepts/resources works

### **Red Light ✗ (Fix Before Launch)**

- ✗ Landmark set still 90% from 2020s (recency bias remains)
- ✗ All resources from same author (bias)
- ✗ Concepts missing prerequisites (cognitive overload risk)
- ✗ Resources don't match learning objectives
